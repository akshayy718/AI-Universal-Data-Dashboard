## `$ what_is_this`

Drop **any** data file вҖ” Excel, CSV, PDF, or Word вҖ” and this dashboard figures out what's inside, builds charts automatically, and lets you have a conversation with your data using **Groq AI**. No setup. No configuration. No SQL.

> Just upload вҶ’ explore вҶ’ ask questions вҶ’ export.

## `$ features --list`

```
вңҰ FILE SUPPORT
  в”ңв”Җв”Җ CSV            вҶ’  Auto-parsed, any structure
  в”ңв”Җв”Җ Excel (.xlsx)  вҶ’  Multi-sheet aware
  в”ңв”Җв”Җ PDF            вҶ’  Table extraction via pdfplumber
  в””в”Җв”Җ Word (.docx)   вҶ’  Table extraction via python-docx

вңҰ AUTO-DETECTION
  в”ңв”Җв”Җ Task / Name column
  в”ңв”Җв”Җ Status column   (Done В· Pending В· In Progress В· Closed...)
  в”ңв”Җв”Җ Due Date column (EDD В· Deadline В· Target В· End Date...)
  в””в”Җв”Җ Category column (Department В· Type В· Module В· Team...)

вңҰ SMART METRICS
  в”ңв”Җв”Җ Total rows В· Completed В· Pending
  в”ңв”Җв”Җ Overdue count В· Due this week В· No due date
  в””в”Җв”Җ Live progress bar with completion %

вңҰ CHARTS (Auto-Generated)
  в”ңв”Җв”Җ Pie  вҶ’ Tasks by Category
  в”ңв”Җв”Җ Bar  вҶ’ Tasks by Status (colour-coded)
  в”ңв”Җв”Җ Pie  вҶ’ Due Date Status Distribution
  в”ңв”Җв”Җ Scatter вҶ’ Task Timeline by Due Date
  в””в”Җв”Җ General Analyzer вҶ’ any other numeric/category columns

вңҰ GROQ AI
  в”ңв”Җв”Җ рҹ“Ҡ Generate a 5-point insight summary
  в”ңв”Җв”Җ рҹҡЁ "What should I focus on?" вҖ” top 3 priorities
  в””в”Җв”Җ рҹ’¬ Free-text Q&A вҖ” ask anything about your data

вңҰ EXPORT
  в”ңв”Җв”Җ Filtered data  вҶ’ CSV
  в”ңв”Җв”Җ Overdue tasks  вҶ’ CSV
  в””в”Җв”Җ Summary report вҶ’ CSV (per category breakdown)
```

### Run locally

```bash
# 1. Clone
git clone https://github.com/akshayy718/AI-Universal-Data-Dashboard.git
cd AI-Universal-Data-Dashboard

# 2. Install
pip install -r requirements.txt

# 3. Run
streamlit run app.py
```

Open вҶ’ `http://localhost:8501`

