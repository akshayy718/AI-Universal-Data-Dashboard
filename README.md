## рҹ“Ң Overview

A **universal AI-powered data dashboard** built with Python and Streamlit. Upload **any** Excel, CSV, PDF, or Word file вҖ” the dashboard automatically detects your columns, builds interactive charts, tracks deadlines, and lets you **chat with your data** using Groq's Llama 3.3 70B model. Zero configuration required.

> рҹ§  Works with any dataset вҖ” project trackers, sales reports, HR data, finance sheets, and more.

---

## вңЁ Features

| Feature | Description |
|---------|-------------|
| рҹ“Ғ **Multi-Format Upload** | CSV, Excel, PDF (tables), Word (tables) |
| рҹ”Қ **Auto Column Detection** | Finds task, status, date, and category columns automatically |
| рҹ“Ҡ **Smart Charts** | Pie, Bar, Timeline, and Scatter вҖ” all auto-generated |
| рҹ“Ҳ **Key Metrics** | Total, Completed, Pending, Overdue, Due This Week |
| вҸ° **Deadline Tracker** | Upcoming deadlines in the next 30 days with urgency colour coding |
| рҹӨ– **Groq AI Q&A** | Ask anything about your data in plain English |
| рҹ”Һ **General Analyzer** | Auto-discovers and charts any extra numeric or category columns |
| рҹ’ҫ **Export** | Download filtered data, overdue tasks, or summary report as CSV |
| вҡҷпёҸ **Column Override** | Manually correct any auto-detected column mapping |
